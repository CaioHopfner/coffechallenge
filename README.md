## Getting Started

### Prerequisites
* Node.js, Vue 2.6, VSCode

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/CaioHopfner/coffechallenge.git
   ```
2. Navigate to the project directory:
   ```bash
   cd repo
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the Project
* Start the development server:
   ```bash
   npm run dev 
   ```
* Open [http://localhost:3000](http://localhost:3000) in your browser.
